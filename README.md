# 🎮 PetVille Adventure

Um jogo Roblox inovador que combina **Adopt Me** com **RPG de Aventura**!

## 🎯 Conceito

Em vez de apenas cuidar de pets, você **aventura com eles**! Cada pet tem habilidades únicas, explore mundos abertos, complete missões, evolua seus pets e domine mini-combates.

## 🔥 Mecânicas Principais

### 1. 🐉 Pets com Habilidades
- **Dragão** → Voar
- **Cão** → Correr rápido
- **Gato** → Saltar alto
- Cada pet tem poderes únicos!

### 2. 🗺️ Mundo Aberto
- **Floresta** 🌲 - Exploração natural
- **Cidade** 🏙️ - NPCs e comércios
- **Caverna Secreta** 🕳️ - Tesouro raro
- **Portais** 🚀 - Mundos diferentes

### 3. 🎯 Sistema de Missões
- Encontrar tesouros 💰
- Ajudar NPCs
- Caçar itens raros
- Ganhar recompensas XP

### 4. ⚔️ Mini Combates
- Pets podem lutar (leve, não violento)
- Sistema de tipo/vantagem
- Ganho de XP
- Evolução através de batalhas

### 5. 🧬 Evolução Real
- Pet muda aparência
- Fica mais forte
- Aprende novas habilidades
- Sistema de leveling

### 6. 🏡 Casa do Jogador
- Decoração customizável
- Treino de pets
- Upgrades especiais
- Armazenamento

## ✨ Features Extra

- 🌦️ Sistema de clima (chuva, neve)
- 🧠 Pets com personalidade
- 🏆 Boss raro no mapa
- 💫 Efeitos visuais incríveis
- 🎵 Música dinâmica
- 🏅 Sistema de conquistas

## 🚀 Como Começar

### No Roblox Studio:
1. Abra Roblox Studio
2. Crie um novo jogo
3. Importe os scripts da pasta `src/`
4. Configure os modelos de pets
5. Teste no Studio!

### Desenvolvimento:
1. Clone este repositório
2. Abra os arquivos Lua em um editor (VS Code com extensão Roblox)
3. Faça suas alterações
4. Teste no Studio
5. Commit e push das mudanças
